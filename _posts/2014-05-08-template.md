## 作用域

模板中所有变量的作用域为元素数据源中的 `data` 值；

```html
<!DOCYPE html>
<html>
    <body>
        <!--嵌套 x-ajax 作为数据源-->
        <x-slider>
            <!--假设返回内容为
            {
                err:0,
                data:[
                    {
                        title:'A',
                        src:'a.png',
                        href:'a.html'
                    },
                    {
                        title:'B',
                        src:'b.png',
                        href:'b.html'
                    }
                ]
            }
            -->
            <x-ajax
                url="https://x.com/data.json" />
            <x-template bind="{ {data}}">
                <ul>
                    <x-template repeat>
                    <li>
                        <a href="{ {href}}">
                            <img src="{ {src}}">
                            <span>{ {title}}</span>
                        </a>
                    </li>
                    </x-template>
                </ul>
            </x-template>
        </x-slider>
    </body>
</html>
```
模板渲染后 `x-slider` 的内部结构为

```html
<!DOCYPE html>
<html>
    <body>
        <x-slider>
            <ul>
                <li>
                    <a href="a.html">
                        <img src="a.png">
                        <span>A</span>
                    </a>
                </li>
                <li>
                    <a href="b.html">
                        <img src="b.png">
                        <span>B</span>
                    </a>
                </li>
            </ul>
        </x-slider>
    </body>
</html>
```
