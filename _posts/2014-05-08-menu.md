## 属性

### `source-selector` **string**

数据源，设置后将自动生成 `menu` 内部结构；

默认数据源结构如下：

    [
        // 通过纯文本实现最简单的功能
        'Action',
        'Another Action',
        'Someone else here',
        // 分隔符
        'divider',
        // 通过对象可以获得更多的功能
        {
            label: 'Open Baidu',
            icon: 'grid',
            href: 'http://www.baidu.com'
        }
    ]
